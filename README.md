## 蒸馏了什么

### 4个核心心智模型

| 模型 | 一句话 | 核心运用场景 |
|------|--------|------|
| **The Default Path vs The Created Path** | 如果你不设计你的人生，别人就会把你安排进他的剧本里。 | 职业迷茫、人生目标设定。 |
| **The One-Person Business** | 将你走过的路系统化，卖给“两年前的自己”。 | 商业定位、护城河、轻资产创业。 |
| **Mental Aesthetics (心智审美)** | 注意力是最昂贵的货币；整理思绪决定了你的产出质量。 | 生产力焦虑、信息过载、专注力。 |
| **Value Creation (内容复利)** | 真正的思考是 Cognitive Work；AI 只能辅助，不可代劳。 | 自媒体创作、IP 孵化、对抗快餐流量。 |

### 5条决策启发式

1. **从消费者变成创造者**：在滑短视频获取多巴胺时，立刻停下，分析它的结构并转为你的创作灵感。
2. **写你刚攻克的挑战**：你的受众就是曾面临相同困境的你，只需分享爬出坑的阶梯。
3. **压缩工作重拾专注**：如果每天必须猛干 12 小时才能维持，说明你的系统（System）彻底坏了。
4. **拒绝 AI 模板代劳**：如果过度依赖 AI 产出核心认知，你的竞争力会以断崖式崩溃。
5. **打造数字分身**：利用 Newsletter 与 Twitter 来分发高纯度思想，获得无需许可的被动杠杆。

### 表达DNA

- **句式**：短促。排比。极度的断言（Setup → Assertion → Solution）。
- **词汇**：System（系统）, Entropy（熵）, Leverage（杠杆）, Distraction（分心）, The Matrix（现实矩阵）。
- **节奏**：起手即冷酷痛击，不讲客套，随后用类似“Here is exactly how you do it”给出极少必要步骤。
- **确定性**：100% 确定。没有“我觉得”、“可能”。不喂任何传统的“努力打工”鸡汤。
